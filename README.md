# 2026 南宁中考择校助手

一个帮助南宁市中考考生和家长根据成绩快速匹配目标学校的工具网站。

## 功能

- **三步输入**：选择学生身份类型 → 选择总成绩等级 → 选择成绩组合
- **智能匹配**：根据成绩组合自动定位全市累计排名
- **三档推荐**：
  - 冲刺学校（排名暂未达到，需努力冲刺）
  - 稳妥学校（排名接近，录取概率较大）
  - 保底学校（排名远超录取线，可作为保底）
- **身份区分**：支持地段生、非地段生、非南宁户籍三种身份，分别匹配定向/非定向录取线

## 数据来源

- 2026年南宁市中考成绩分布（一分一档）
- 2025年示范性高中录取最低成绩表（非定向 + 定向）
- 2025年示范性高中高考特控率

## 使用方法

1. 直接打开 `index.html` 即可使用，无需服务器
2. 选择学生身份类型（地段生 / 非地段生 / 非南宁户籍）
3. 点击总成绩等级（A+ ~ E）
4. 选择具体成绩组合
5. 点击「开始匹配」查看推荐结果

## 技术栈

- 纯前端 HTML + CSS + JavaScript
- 数据内嵌于页面中，无需后端
- 响应式设计，支持手机端

## 部署

本项目为纯静态页面，可通过以下方式部署：

- **GitHub Pages**：将仓库推送到 GitHub，在 Settings > Pages 中启用
- **Vercel / Netlify**：直接导入仓库即可
- **任意静态服务器**：将 `index.html` 放到任意 Web 服务器目录下
